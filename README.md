# project_template
The "project template" helps you to run your workspace fast and ease !

## Template includes:

- webpack/webpack-dev-server
- pug
- sass
- jquery
- cssnano
- autoprefixer
- reset-css

## Scripts:

  ### start a dev-server: 
  ```sh
    npm run start
  ```
    
  ### build a project:
  ```sh
    npm run build
  ```
